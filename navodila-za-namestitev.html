1. Namestitev podatkovne baze mongodb

Za začetek dela potrebujemo zbirko podatkov. Obrniti se morate na https://www.mongodb.com/ in se prijavite. Uporabite lahko spletno mesto ali namestite aplikacijo. Nato lahko ustvarite nov Cluster ali se povežete z obstoječem. V drugem primeru vas mora lastnik zbirke podatkov dodati kot uporabnika.

POZOR: Podatkovna baza je vezana na vaš IP. Če jo želite odpreti na novem mestu, boste morali iti na spletno stran mongo.db in dodati nov naslov ip. Več podrobnosti si lahko preberete na: https://www.mongodb.com/docs/atlas/.

2. Namestitev git

Obiskati morate spletno stran https://git-scm.com/downloads in namestite različico, ki jo potrebujete za svoj operacijski sistem za namestitev programa git. Za več podrobnosti lahko obiščete: https://git-scm.com/doc and read the documentation.

3. Namestitev namizne aplikacije in vsega, kar potrebujete za njeno delovanje

Za namestitev namizne aplikacije potrebujete naslednje:

  • Intelij Idea
  • JDK (version 18)

Prenesete jo lahko iz https://www.jetbrains.com/ (Intelij Idea) in https://www.oracle.com/cis/java/technologies/downloads/ (JDK). Za več podrobnosti si lahko ogledate uradno spletno stran https://www.jetbrains.com/lp/compose-multiplatform/.

Po namestitvi boste morali klonirati repozitorij https://github.com/Syntoxic-RIT/DesktopApp.

To lahko storite na dva načina: s programom Intelij Idea ali s pomočjo git. Podrobna navodila za namestitev repozitorija so na voljo na: https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository?platform=windows.

Seznanite se z aplikacijo.

POZOR: Čez nekaj časa se lahko zgodi, da dependencies ne bodo delovale, ker so zastarele. V tem primeru projekta ne boste mogli zgraditi. Če želite rešiti to težavo, pojdite v datoteko build-gradle in jih posodobite na najnovejšo različico.

4. Namestitev spletne aplikacije in vsega, kar potrebujete za njeno delovanje

Za namestitev spletne aplikacije potrebujete aplikacijo, ki podpira node.js in react. Priporočamo namestitev VS Code. Prenesete jo lahko s spletne stranihttps://code.visualstudio.com/. Prenesti boste morali tudi https://nodejs.org/en. Nato v ukazno vrstico zapišite naslednje ukaze:

npm install
npm install react
npm install nodemon

Po namestitvi boste morali klonirati repozitorij nahttps://github.com/Syntoxic-RIT/WebApp.

To lahko storite na dva načina: s programom VS code ali s pomočjo git. Podrobna navodila za namestitev repozitorija so na voljo na: https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository?platform=windows. The latest version of the application is on the development branch.

Da bo aplikacija delovala, morate namestiti zbirko podatkov, za kar sledite točki 1.

Če želite zagnati aplikacijo, morate z desno tipko miške klikniti na mapo backend, izbrati možnost Odpri v integriranem terminalu in uporabiti ukaz npm start. Nato ponovite korake za mapo frontend. Uporabite lahko tudi npm run dev ukaz

Zaradi izdaje novih različic se lahko zgodi, da je treba kodo posodobiti.

Več informacij o podobnih aplikacijah: https://www.freecodecamp.org/news/how-to-create-a-react-app-with-a-node-backend-the-complete-guide/.

5. Namestitev spletnega mesta na strežnik. Delo z Azure

Na voljo sta dve možnosti: spletno mesto gostite sami ali do njega dostopate prek ssh. Najprej se morate registrirati na https://azure.microsoft.com.

Izberete lahko način plačila in konfiguracijo virtualne naprave. Za študente je na voljo brezplačno poskusno obdobje. Za povezavo prek ssh vas mora nekdo dodati kot uporabnika in dodati geslo. Nato se lahko povežete na naslednji način:

Odprite Windows Powershell. Vnesite ukaz ssh Username@20.160.29.130, nato vnesite geslo. Spletna stran bo na voljo na naslovu http://20.160.29.130:3000/.

Če želite preizkusiti vse funkcije na spletni strani, morate v brskalniku Chrome vstopiti na stran z naslovom chrome://flags/#unsafely-treat-insecure-origin-as-secure. Nato v prazno polje vnesite naslov url od spletne strani na azure http://20.160.29.130:3000/
